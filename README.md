# MoviesChallenge

Application that displays the list and details of popular movies and upcoming movies using TheMovieDB API.

# Features
* MVVM Architecture
* Auto switch
* Pull to refresh
* Information on popular and upcoming movies

# Pods

```yaml
  pod 'Alamofire'
  pod 'Kingfisher'
  pod 'SnapKit'
```
